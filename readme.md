# Vanessa Streck | Escritório de Advocacia

Landing Page profissional para o escritório de advocacia Vanessa Streck, com foco em Direito Previdenciário e Civil. Uma interface moderna, responsiva e otimizada para conversão (CRO).

## ⚡ Quick Start

Apenas abra o `index.html` em qualquer navegador ou use um servidor local (como Live Server):

```bash
# Se usar npm/node
npx serve .
```

## ✨ Features

- **Design Premium:** Cores bordô e preto com tipografia Newsreader e Inter.
- **Responsível:** Otimizado para Desktop, Tablet e Mobile.
- **Animações Fluidas:** Scroll Reveal dinâmico com Intersection Observer.
- **Integração Web3Forms:** Formulário de contato funcional com envio assíncrono.
- **Widget WhatsApp:** Acesso rápido ao atendimento jurídico via botão flutuante.
- **Proteção Anticópia:** Medidas de segurança contra inspeção de código e salvamento de imagens.

## 🛡️ Segurança e Privacidade

Este projeto implementa medidas ativas de proteção de conteúdo:
- Bloqueio de clique direito (Menu de contexto).
- Bloqueio de atalhos de desenvolvedor (F12, Ctrl+U, Ctrl+Shift+I).
- Imagens inarrastáveis (Anti-drag).
- **Sem Cookies:** O site não utiliza cookies de rastreamento no momento.
